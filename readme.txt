1.布局，页面中间的主体部分，分为左右两栏，左侧固定宽度，内容为Canvas画布，可以拖动其中两个控制点；右侧自适应宽度，包括控制按钮，函数效果展示，输入控件，代码显示，以及保存的结果。

2.Canvas绘制功能，将其封装成一个js文件，留有canvas接口，包括控制点当前坐标，画布初始化，画布绘制等功能。

3.在main.js中，主要对控制按钮以及输入控件等进行时间监听，以及对用户定义的函数进行保存、删除。

4.在函数效果展示中，通过执行生成的代码来控制滑块左右滑动。

5.输入控件可以输入两个控制点的x，y坐标，其中x坐标范围为0-1，y不限制，间隔均为0.1；也可以输入时间，范围0-10s，间隔0.5s

6.代码显示部分，会根据之前的操作进行及时的刷新。

7.代码保存功能是利用localStorage完成，允许用户自定义名称，点击该保存的结果，就会刷新当前的canvas；点击并确认删除后，localStorage存储的结果也可以删除。
